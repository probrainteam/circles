# Circles - 동아리 회원 관리 웹 애플리케이션

<br/>

### 🛠 Tech Stack
- React
- Redux
- React-query
- emotion.js
- Figma

<br/>

### 📋 주요 기능
- Todolist / 캘린더
- 학사 공지
- 구글 폼을 통한 동아리 회원 신청 승인
- 회원 데이터 시각화
- 단체 카카오톡 전송
- 다양한 상황에 맞는 클립보드 제공
- 회원 데이터 엑셀로 내보내기
